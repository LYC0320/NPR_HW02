Run LayoutCreation.exe to generate a texture and then run CG_HW03.exe in Rendering folder to show the result.

CG_HW03.exe:

mouse指令
按住左鍵拖曳可移動視角

keybooard 指令(全小寫)
w/s/a/d/q/e: camera 根據視角 前後左右上下 移動
t/g/f/h/r/y: light source 根據視角 前後左右上下 移動
u/j/h/k/y/i: ball item 根據視角 前後左右上下 移動
7/4/8/5/9/6: ball item 物體座標 +x -x +y -y +z -z轉動
z:move light source to front of camera
x:move ball to front of camera
c:reset all pose
b:toggle bump mapping

The effect of height mapping is not satisfactory because of the low number of triangles.
